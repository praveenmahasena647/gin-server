# gin-server

This is a simple API server which was written in [Golang](https://go.dev/) with the perpose of learning [gin-go](https://gin-gonic.com/).
 
## Endpoints

* `GET /`
* `GET /:id`
* `POST /`
* `PUT /:id`
* `DELETE /:id`
* `DELETE /`

## Development
```sh
git clone https://github.com/praveenmahasena647/gin-server.git . && go run .
```
